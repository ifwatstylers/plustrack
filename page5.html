FAQ

What is the PLUSTrack Program?

PLUSTrack program is created for fleet operators that provides convenience, saving, effective management and control for fleet operations.

How does PLUSTrack work?

It is a prepaid (debit) toll card with company name and vehicle registration number imprinted on it. Together with the PLUSTrack Reload Terminal, fleet operators can top up the PLUSTrack cards at their ease and convenience without having to incur any reload fees. Furthermore, the toll transaction history can be viewed from the dedicated Touch 'N Go portal while the usage of the Reload Terminal can be accessed by a web-based system provided.

How to apply for PLUSTrack?

Register your interest via email to b2b.plustrack@plus.com.my. You will need to fill in a registration form and submit the requested documents for processing.

How many cards are allowed for one vehicle?

One card per vehicle. However there is no limit to the number of cards per company.

Who is entitled to the PLUSTrack program?

The program is targeted for fleet managers/owners or driver-owners who own Class 2 and/or Class 3 commercial vehicles.

What are the benefits of PLUSTrack?

Subscribers will enjoy more efficient day-to-day fleet management, self-serviced transaction tracking, customized corporate branding on PLUSTrack card as well as monitoring toll expenditures for better fleet management.

Is there any charges to be paid for subscription?

There are no charges for the membership subscription. However, there is a rental fee for the Reload Terminal if required.

What is the package offer and charges?

Pakej A
PLUSTrack card = RM 10.00 / card
Reload Terminal Rental = RM 60.00 / month
Reload Terminal Insurance = RM 40.00 / year

Pakej B
PLUSTrack card = RM 10.00 / card. No Reload Terminal.

What is the minimum number of vehicles to apply?

There is no minimum number of vehicles to apply.

Can the number of subscription be increased from time to time?

Yes. Subscribers may add number of vehicles under PLUSTrack from time to time.

Can PLUSTrack card be used for toll?

Yes. PLUSTrack card can be used for toll only

Can PLUSTrack card be used for parking?

No. PLUSTrack card can be used for toll only

Can PLUSTrack card be used for retail?

No. PLUSTrack card can be used for toll only.

Can PLUSTrack card be used for transit?

No. PLUSTrack card can be used for toll only.

Can PLUSTrack card be used for other highway?

Yes. PLUSTrack can be used at all highway network in Peninsular Malaysia.

How to reload the PLUSTrack card?

PLUSTrack card can be reloaded in 2 ways:
1- Using Reload Terminal (for those who rent the terminal)
2- All Touch 'N Go customer service centre, ATMs, Self-Service Kiosks, Petrol Stations and any other authorized points that offer Touch n Go reload

How does the reload terminal's function?

The reload terminal is used to top-up the Touch n Go cards, provided that there is credit inside the reload terminal.

Does the Reload Terminal impose a reload fee for every reload transaction?

No. The reload terminal does not impose a reload fee for every reload transaction.

How to monitor the transaction history?

The subscriber may view the transactions online via the Touch 'N Go web portal provided.
